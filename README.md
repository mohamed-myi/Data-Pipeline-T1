# Data Pipeline

A Python FastAPI-based data pipeline application for processing, transforming, and managing data workflows.

## Setup

1. Clone the repository
2. Create a virtual environment: `python -m venv venv`
3. Activate: `source venv/bin/activate` (Mac/Linux) or `venv\Scripts\activate` (Windows)
4. Install dependencies: `pip install -r requirements.txt`
5. Copy `.env.example` to `.env` and configure database connection
6. Run: `python main.py`

The API will be available at `http://localhost:8000`

## Testing

Run tests with: `pytest`
